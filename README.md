# PUNC
Repositório reponsável pelo projeto PUNC do Projeto Integrador III do IFC Campus Concórdia.

Será um projeto focado na notificação urbana da coleta de lixo baseando-se em informações em tempo real como a localização, tanto do cidadão que receberá a notificação como do motorista do caminhão que enviará a sua localização para fazer o envio próprio.

Todas as ações criadas neste repositório são baseadas no pré-projeto da PUNC, não disponível publicamente.
